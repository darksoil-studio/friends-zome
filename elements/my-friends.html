Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <my-friends> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/friends-zome/dist/elements/my-friends.js'
  1. Use it in the html side of your web-app like this:
html
<my-friends>
</my-friends>

WARNING
